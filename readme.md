# Hero.IO – Free Apps Download Platform

**Hero.IO** is a modern, responsive web application that lets users discover, explore, and download the most popular applications in one beautiful place.

Live URL: https://heroio-knox.surge.sh  


## Key Features

- Clean app detail pages with hero banners  
- Fully responsive design – looks perfect on mobile, tablet, and desktop    
- Latest app updates & patch   
- "Install Now" buttons that let's you install apps    
- Beautiful UI with good UX  

## Tech Stack & npm Packages Used

 1. `react` + `react-dom` Core framework                           
 2. `react-router` Client-side routing & protected routes    
 3. `tailwindcss` + `daisyui` + `react-toastify`  Design and styling                     
 4. `lucide-react` + `react-icons` Beautiful icons          
 5. `surge` deployment
